## Markdown
### Списки
Списки бывают разные:
1. Звездочки
  * Используются через *
2. Плюсики
  + Используется через +
3. Минусы
  - Используется через -
4. C цитатой
    > Вот тут цитата
    > великого человека

При этом вверху - нумерованный, а все ненумерованные списки выглядят одинаково
